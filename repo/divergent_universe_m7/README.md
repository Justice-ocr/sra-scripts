# 差分宇宙（三月七小助手）

调用 [March7thAssistant（三月七小助手）](https://github.com/moesnow/March7thAssistant) 执行差分宇宙，支持在 SRA 脚本配置页直接修改三月七的差分宇宙相关参数。

## 前置要求

- 已安装 March7thAssistant（三月七小助手）
- 三月七至少完成过一次配置（config/config.json 存在）

## 配置说明

在 SRA 任务页选中此自定义任务后，点击「修改脚本配置」：

| 配置项 | 说明 | 默认值 |
|--------|------|--------|
| 三月七安装目录 | March7th.exe 所在文件夹 | `C:\Program Files\March7thAssistant` |
| 难度等级 | 差分宇宙难度（1-5 层） | `4` |
| 运行次数 | 执行次数，`-1` 为无限循环 | `1` |
| 队伍编号 | 使用的队伍（1 起） | `1` |
| 使用支援角色 | 是否使用好友支援 | `false` |
| 支援角色名 | 指定支援角色（留空选第一个） | 空 |
| 使用秘技 | 进入战斗前使用秘技 | `true` |
| 秘技战斗 | 用秘技直接开启战斗 | `true` |
| 优先祝福索引 | 优先选第几个祝福（0=随机） | `1` |
| 最大可损失祝福数 | 超过此数重开（0=不限） | `0` |
| 超时时间 | 等待三月七完成的最长秒数 | `600` |

## 工作原理

1. 读取 SRA 中配置的参数
2. 备份三月七 `config/config.json` 中的差分宇宙字段
3. 将 SRA 参数写入三月七配置
4. 以 `--auto-run --task universe --exit-after-done` 参数启动三月七
5. 等待三月七进程退出
6. 恢复三月七原始配置

## 注意事项

- 运行期间不要手动操作三月七
- 如果三月七版本更新导致命令行参数变化，请提交 Issue
- 超时时间建议设置为单次运行预期时长的 1.5 倍
